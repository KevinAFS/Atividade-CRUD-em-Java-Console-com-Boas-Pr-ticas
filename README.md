# Sistema de Gerenciamento de Tarefas (CRUD em Java - Console)

## Descrição do Projeto

Este projeto consiste em um **sistema simples de gerenciamento de tarefas**, desenvolvido em **Java**, com execução no **console**. O objetivo é aplicar os conceitos de **Programação Orientada a Objetos (POO)** e boas práticas de desenvolvimento, como **separação em camadas (Model, Repository, Service e App)** e **responsabilidade única**.

O sistema permite realizar todas as operações CRUD:

- **Criar (Cadastrar)**
- **Ler (Listar)**
- **Atualizar**
- **Excluir**

Cada tarefa possui os seguintes atributos:

- `id` (inteiro, autoincrementado)
- `título`
- `descrição`
- `data de conclusão` (`LocalDate`)
